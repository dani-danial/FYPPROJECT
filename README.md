RUNNING TRACKER: AI-Integrated Mobile Application with Telegram SOS 🏃‍♂️🤖

Running Tracker is a comprehensive health and fitness platform developed as a Final Year Project for the Netcentric Computing (CDCS251) program at UiTM Melaka Kampus Jasin. The application bridges the gap between traditional fitness tracking and real-time community safety through an integrated AI Coach and a specialized Telegram Bot for emergency response.  

🚀 Key Features
1. Intelligent Onboarding & Leveling
AI Assessment: A conversational chatbot collects data on your running style, usual distance, and weekly frequency.

Dynamic Categorization: The backend processes user inputs to assign a fitness level: LOW, MID, or HARD.

Smart Event Discovery: Suggests running events based on the user's current level, providing full event details and venue locations.

2. Strava-Style Activity Tracking
Live GPS Tracking: Real-time location detection for outdoor runs.

Performance Metrics: Tracks total distance (km), duration, and average pace.

Activity History: Syncs data to the user profile to track total kilometers and run counts.

3. Integrated AI Coach
Context-Aware Advice: An AI coach analyzes user physical stats (Weight, Height) to provide personalized running schedules and dietary plans.

Interactive Chat: Directly chat with the AI for immediate fitness guidance.

4. Community & Social Networking
Social Feed: Share posts, follow other runners, and see friend activity in a real-time feed.

Direct Messaging: Built-in chat functionality to communicate with followers.

Running Groups: Join existing communities or create your own running group to stay motivated.

5. Netcentric Safety System (Telegram SOS) 🆘
A standout safety feature designed for solo runners. In the event of a fall or injury, the app triggers a high-priority alert to a Telegram bot and group.

🛠️ Tech Stack
Mobile Frontend: Java (Android Studio)

Backend: Node.js / Express

Real-time Database: Firebase (User profiles, social feed, and chat)

APIs: Telegram Bot API, Google Maps SDK (Location tracking), Retrofit (Networking)

Image Loading: Glide
